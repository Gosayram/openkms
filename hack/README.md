# Hack Scripts

Utility scripts for project maintenance.

## Scripts

### add-copyright.sh
Adds copyright header to Go files that don't have it.

```bash
./hack/add-copyright.sh
```

### check-copyright.sh
Checks that all Go files have correct copyright header.

```bash
./hack/check-copyright.sh
```

### update-copyright.sh
Updates copyright year in all files to current year.

```bash
./hack/update-copyright.sh
```

## Usage

These scripts can be run manually or integrated into CI/CD pipelines to ensure copyright headers are maintained.
